NC NEWS

I created a full stack news aggregation board, where users can view articles from different topics, sort/filter articles, vote on articles, post comments and vote on comments. It employs a RESTful API that follows an MVC architecture pattern. The API was developed using Node.js, Express.js, PostgreSQL and Knex. The front end was developed using React.js, HTML5 and CSS.
